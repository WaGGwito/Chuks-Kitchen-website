# 🍽️ Chuks Kitchen - website

Welcome to the **Chuks Kitchen** project! This is a modern, high fidelity web application designed to showcase authentic Nigerian cuisine. The project focuses on a Mobile First experience with a clean interface and highly responsive layout.

### ✨ Key Features

* **Responsive Hierarchy**: A layout that shifts from a side-by-side desktop view to a vertically optimized mobile stack.

* **Adaptive Hero Section**: Utilizes responsive image delivery to serve high-quality assets tailored for both desktop and mobile screen ratios.

* **Modern UI/UX**: Built with a warm, brand-specific color palette and thumb friendly button placements to optimize the mobile ordering experience.

### 🛠️ Tech Stack

* **React**: For building the component-based user interface.
* **Tailwind CSS**: For utility first styling and precise responsive design.
* **Vite**: For a fast development environment and optimized production builds.
* **React Router**: To handle seamless navigation across the application.

### 🚀 Getting Started

To run this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone [your-repository-url]
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Start the development server:**
    ```bash
    npm run dev
    ```
4.  **View the project:**
    Open your browser to `http://localhost:5176`.
